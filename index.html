🚀 Low-SWaP-C Flight Controller

Senior Capstone Project

Welcome

A Low Size, Weight, Power, and Cost (SWaP-C) flight controller designed for drone and embedded aerospace systems.

📸 Current Design

🧠 What It Does

  • 📦 Low SWaP-C: Compact and efficient
  • 📈 Sensor Fusion: IMU + Magnetometer + Barometer
  • 🔌 Modular I/O: UART, I2C, PWM
  • 🧭 Navigation Ready: GPS and streaming
  • 🎛️ Open Source Firmware (STM32)

🔧 How It Works

🛠️ Hardware Overview

  • MCU: STM32F405
  • IMU: Honeywell I400 MEMS
  • Power: Onboard 3.3V and 5V regulation

🧬 Software Features

  • Written in C/C++ (CircuitPython-compatible)
  • Real-time sensor polling/filtering
  • ESC signal generation/control logic
  • Modular firmware

🎯 Use Cases

  • Autonomous drone systems
  • Flight controller dev kits
  • Lightweight UAS prototyping
  • Educational use in embedded/control systems
