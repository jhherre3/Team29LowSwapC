This project presents a Low SWaP-C flight controller optimized for lightweight UAV systems and embedded applications.

  • MCU: STM32F405
  • IMU: Honeywell I400
  • Barometer, GPS integration
  • UART, I2C, PWM interfaces

What is an IMU?

An Inertial Measurement Unit detects acceleration and angular velocity.

What is a barometer used for?

It measures atmospheric pressure to estimate altitude.

How does GPS help in drones?

It provides real-time location for navigation and waypoint following.

How does a drone work?

By using propellers, sensors, and a flight controller to remain balanced and follow commands.

What is SWaP-C?

Size, Weight, Power, and Cost — critical in aerospace design.

Why use STM32?

It's reliable, efficient, and powerful for embedded systems.

What firmware is used?

C/C++ firmware with real-time control loop.

What is sensor fusion?

Combining sensor data to produce reliable orientation and position estimates.
